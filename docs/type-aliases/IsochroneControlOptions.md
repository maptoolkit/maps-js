# IsochroneControlOptions

> **IsochroneControlOptions** = `object`

Options for configuring the [IsochroneControl](../classes/IsochroneControl.md).

## Properties

### fitBounds?

> `optional` **fitBounds**: `boolean` \| [`FitBoundsOptions`](FitBoundsOptions.md)

Whether the map bounds should fit the isochrone area. Alternatively accepts [FitBoundsOptions](FitBoundsOptions.md) object.

#### Default Value

`true`

***

### range?

> `optional` **range**: `number`

Time used for isochrone calculation. Value in minutes.

#### Default Value

`10` minutes

***

### type?

> `optional` **type**: `IsochroneType`

Routing type used for isochrone request. Default value is `foot`.

#### Default Value

`10` minutes
