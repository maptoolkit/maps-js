# AddLayerObject

> **AddLayerObject** = `LayerSpecification` \| `Omit`\<`LayerSpecification`, `"source"`\> & `object` \| [`CustomLayerInterface`](../interfaces/CustomLayerInterface.md)

Specifies a layer to be added to a Style. In addition to a standard LayerSpecification
or a [CustomLayerInterface](../interfaces/CustomLayerInterface.md), a LayerSpecification with an embedded SourceSpecification can also be provided.
